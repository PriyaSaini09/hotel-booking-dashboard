Hotel Booking Cancellation Dashboard
Overview

This project presents an interactive Excel dashboard analyzing hotel booking data.
It provides insights into booking trends, cancellation behavior, guest types, and room preferences.
Key Insights
Total Bookings: 119,391
Total Cancellations: 44,224
Cancellation Rate: 37%
City Hotels have higher bookings compared to Resort Hotels
Peak bookings observed during mid-year months
Features
Interactive dashboard with Year slicer (2015–2017)
KPI cards for quick insights
Guest type analysis (Couples, Family, Single)
Room type analysis (Desired vs Undesired)
Monthly booking trends visualization
Tools Used
Microsoft Excel
Pivot Tables
Data Visualization
Project Purpose
This dashboard helps businesses understand customer booking patterns and identify key factors influencing cancellations, enabling better decision-making.
